## AI时代的趋势
1. **DAU 已过时。** AI 产品每多服务一个用户，就会多一份推理成本。微信是网状拓扑，ChatGPT 是星状拓扑，而网状拓扑才有网络效应
2. **不再有工具到平台的路径。** 当 AI 比人更能帮人的时候，社区的价值基础就坍塌了。大模型公司可以定义生态、制定标准。
3. **Agent 是SAAS 的新主人。** Agent 对 API 的调用量成指数级增长。过去 SAAS 的客户是企业或消费者，现在 Agent 可以自主学习、效率提升百倍，未来最大的客户是 Agent。
4. ==不要服务人，服务 Agent。==
5. AI 时代追求结果交付效率，注意力经济追求的是用户停留时长。
6. 把 API 做好，把文档写清楚，把协议对接好。全世界的 Agent 都会找到你。
## Agent时代的现象
1. 算力的马太效应。最好的模型定价更贵了，token 费用还在上升。更好的算力意味着更好、更快的结果。
2. ==要做能疯狂燃烧 token 的事和产品==。AI Coding、Agent 和 Video 是今天燃烧 token 最快的三类产品。
3. 怎么让 Agent 用的爽？API 稳不稳定、文档清不清楚、返回结果准不准确。怎么让 Agent 第一个知道你，用了之后离不开你？先被发现，再被依赖。
4. ==Agent 有能力、理性和耐心，人类有欲望、情感和想象==。人类的价值不是亲自干活，是决定干什么、为什么干。
